# 多页对话系统使用说明

## 📖 功能概述

对话系统已支持多页对话配置，点击下一步可以翻页，所有对话页显示完后可以显示选项。

---

## 🎯 数据配置

### DialogData 结构

```csharp
[CreateAssetMenu(fileName = "NewDialogData", menuName = "Data/DialogData")]
public class DialogData : ScriptableObject
{
    [Header("NPC 信息")]
    public string npcName;
    
    [Header("对话内容")]
    [Tooltip("多页对话内容，每行代表一页")]
    [TextArea(3, 10)]
    public List<string> dialogPages = new List<string>();
    
    [Header("选项（可选）")]
    [Tooltip("对话结束后的选项列表")]
    public List<string> dialogOptions = new List<string>();
}
```

### 配置步骤

1. **创建 DialogData**
   - 在 Project 窗口右键 → Create → Data → DialogData
   - 命名文件（如：`NPC_Merchant_Dialog`）

2. **配置对话内容**
   ```
   NPC 名称：商人约翰
   
   对话页数（dialogPages）：
   [0] "你好，勇敢的冒险者！欢迎来到我的商店。"
   [1] "我这里有各种各样的商品，快来看看吧。"
   [2] "如果你有任何问题，随时可以问我。"
   
   选项（dialogOptions）：
   [0] "查看商品"
   [1] "离开商店"
   [2] "询问任务"
   ```

3. **添加到 NPC**
   - 选中场景中的 NPC GameObject
   - 在 NPC 组件的 `dialogData` 字段拖入创建的 DialogData

## 🔄 对话流程

### 完整流程

```
玩家点击 NPC
    ↓
移动到 NPC 身边
    ↓
打开对话框
    ↓
显示第 1 页对话
    ↓
玩家点击下一步
    ↓
显示第 2 页对话
    ↓
玩家点击下一步
    ↓
...
    ↓
显示最后一页对话
    ↓
玩家点击下一步
    ↓
检查是否有选项
    ├─ 有选项 → 显示 Selection 面板
    └─ 无选项 → 关闭对话框
```

### 代码流程

```csharp
DialogManager.StartDialog(dialogData)
    ↓
显示 dialogPages[0]
    ↓
HandleDialogClick() → NextPage()
    ↓
currentPageIndex++
    ↓
检查：currentPageIndex < dialogPages.Count ?
    ├─ 是 → 显示 dialogPages[currentPageIndex]
    └─ 否 → ShowSelection()
        ↓
        检查：dialogOptions.Count > 0 ?
        ├─ 是 → 显示 Selection 面板
        └─ 否 → EndDialog()
```

## 🎨 Selection 面板

### 当前实现

```csharp
private void ShowSelection()
{
    if (currentDialog.dialogOptions.Count > 0)
    {
        // 显示选项数量（临时实现）
        string optionText = $"共有 {currentDialog.dialogOptions.Count} 个选项";
        currentPanel.Show(optionText);
        
        // 显示 Selection 面板
        currentPanel.SetSelectionVisible(true);
    }
    else
    {
        EndDialog();
    }
}
```

## 🔧 代码接口

### DialogManager

```csharp
// 开始对话
DialogManager.Instance.StartDialog(dialogData);

// 强制关闭对话
DialogManager.Instance.ForceClose();

// 检查是否有对话正在进行
bool isTalking = DialogManager.Instance.IsDialogActive;
```

### DialogData（Unity 编辑器中配置）

```csharp
// 添加对话页
dialogData.dialogPages.Add("第一页内容");
dialogData.dialogPages.Add("第二页内容");

// 添加选项
dialogData.dialogOptions.Add("选项 1");
dialogData.dialogOptions.Add("选项 2");
```

## 🐛 注意事项

### 1. DialogData 配置
- ✅ `dialogPages` 至少需要有一页内容
- ⚠️ 如果 `dialogPages` 为空，不会显示任何内容
- ✅ `dialogOptions` 可以为空（直接关闭对话）

### 2. NPC 配置
- ✅ 确保 NPC GameObject 上有 NPC 组件
- ✅ 确保 NPC 组件的 `dialogData` 字段已赋值

### 3. 对话流程
- ✅ 点击下一步会跳过当前打字效果，立即显示完整文字
- ✅ 文字显示完后点击下一步会进入下一页
- ✅ 所有页显示完后会显示选项或关闭对话

### 4. 外部关闭
- ✅ 点击地面可以关闭对话框（会清理状态）
- ✅ 关闭后可以再次与 NPC 对话

## 📊 数据结构对比

### 旧版（单页）
```csharp
public string dialogContent;  // 只能有一页
public List<string> dialogOptions;
```

### 新版（多页）
```csharp
public List<string> dialogPages;  // 支持多页
public List<string> dialogOptions;
```

### 迁移指南

**如果你有旧的 DialogData：**

1. 打开旧的 DialogData 资产
2. 将 `dialogContent` 的内容复制到 `dialogPages[0]`
3. 删除 `dialogContent` 字段（或保留作为备用）

**代码迁移：**

```csharp
// 旧代码
currentPanel.Show(data.dialogContent);

// 新代码
if (data.dialogPages.Count > 0)
{
    currentPanel.Show(data.dialogPages[0]);
}
```
