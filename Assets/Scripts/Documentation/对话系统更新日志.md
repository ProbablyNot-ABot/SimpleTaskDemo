## 🎯 更新内容

### 1. DialogData 数据结构更新

#### 新增字段

**DialogCondition（对话条件）**
```csharp
public class DialogCondition
{
    public bool hasCondition = false;      // 是否有条件
    public string questId;                 // 任务 ID
    public QuestState requiredState;       // 需要的任务状态
}
```

**DialogAction（对话动作）**
```csharp
public class DialogAction
{
    public bool hasAction = false;         // 是否有动作
    public ActionType type;                // 动作类型
    public string questId;                 // 任务 ID
    public GameObject shopPrefab;          // 商店预制体 ⭐ 新增
}
```

**DialogOption（对话选项）**
```csharp
public class DialogOption
{
    public string optionText;              // 选项文字
    public string nextNodeId;              // 跳转节点 ID
    public DialogCondition condition;      // 选择条件
    public DialogAction action;            // 选择后的动作
    public string description;             // 选项说明
}
```

**DialogNode（对话节点）**
```csharp
public class DialogNode
{
    public string nodeId;                  // 节点唯一标识
    public string speakerName;             // 说话者
    public List<string> content;           // 对话内容（多页）
    public List<DialogOption> options;     // 选项列表
    public DialogCondition enterCondition; // 进入条件
    public string description;             // 节点说明
}
```

**DialogData（对话数据容器）**
```csharp
public class DialogData
{
    // 新字段
    public string dialogId;                // 对话唯一标识
    public string npcName;                 // NPC 名称
    public string description;             // 对话描述
    public List<DialogNode> nodes;         // 所有对话节点
    public string startNodeId;             // 起始节点 ID
    public bool allowRepeat;               // 是否允许重复
    public string repeatNodeId;            // 重复对话节点 ID
    
    // 向后兼容字段（旧版）
    public List<string> dialogPages;       // 旧的对话页面列表
    public List<string> dialogOptions;     // 旧的选项列表
}
```

#### 新增枚举

**QuestState（任务状态）**
```csharp
public enum QuestState
{
    NotStarted,     // 未开始
    InProgress,     // 进行中
    Completed,      // 已完成
    Failed          // 失败
}
```

**DialogAction.ActionType（动作类型）**
```csharp
public enum ActionType
{
    None,           // 无动作
    StartQuest,     // 开始任务
    CompleteQuest,  // 完成任务
    OpenShop,       // 打开商店 ⭐ 新增
}
```

### 2. DialogManager 更新

#### 新增功能

1. **自动判断数据格式**
   - 检测是否使用新的 `nodes` 字段
   - 自动选择分支对话系统或简单对话系统
   - 保持向后兼容

2. **分支对话系统**
   - `StartBranchDialog()` - 开始分支对话
   - `NextBranchPage()` - 进入下一页
   - `ShowBranchSelection()` - 显示选项（带条件过滤）
   - `HandleBranchOptionSelected()` - 处理选项选择
   - `GotoNode()` - 跳转到指定节点

3. **条件检查**
   - `CheckCondition()` - 检查任务状态条件
   - 自动过滤不满足条件的选项
   - 支持节点进入条件检查

4. **动作执行**
   - `ExecuteAction()` - 执行对话动作
   - 支持 `StartQuest` - 开始任务
   - 支持 `CompleteQuest` - 完成任务
   - 支持 `OpenShop` - 打开商店 ⭐ 新增

5. **简单对话系统（向后兼容）**
   - `StartSimpleDialog()` - 开始简单对话
   - `NextSimplePage()` - 进入下一页
   - `ShowSimpleSelection()` - 显示选项
   - `HandleSimpleOptionSelected()` - 处理选项选择

#### 状态管理

新增状态变量：
```csharp
private DialogNode currentNode;        // 当前对话节点
private int currentOptionIndex = -1;   // 当前选项索引
```

### 3. 新增功能特性

#### 3.1 分支对话结构

支持树状对话结构，每个节点可以：
- 包含多页对话内容
- 有多个选项供玩家选择
- 设置进入条件
- 执行进入动作
- 跳转到其他节点

#### 3.2 条件系统

**支持的条件类型：**
- ✅ 任务状态检查（QuestState）
  - NotStarted - 未开始
  - InProgress - 进行中
  - Completed - 已完成
  - Failed - 失败

**使用示例：**
```csharp
// 只有完成任务后才显示此选项
option.condition = new DialogCondition
{
    hasCondition = true,
    questId = "kill_goblins",
    requiredState = QuestState.Completed
};
```

#### 3.3 动作系统

**支持的动作类型：**

1. **StartQuest** - 开始任务
```csharp
option.action = new DialogAction
{
    hasAction = true,
    type = DialogAction.ActionType.StartQuest,
    questId = "kill_goblins"
};
```

2. **CompleteQuest** - 完成任务
```csharp
option.action = new DialogAction
{
    hasAction = true,
    type = DialogAction.ActionType.CompleteQuest,
    questId = "kill_goblins"
};
```

3. **OpenShop** - 打开商店 ⭐ 新增
```csharp
option.action = new DialogAction
{
    hasAction = true,
    type = DialogAction.ActionType.OpenShop,
    shopPrefab = shopPrefab  // 拖入商店预制体
};
```

#### 3.4 重复对话

当节点进入条件不满足时：
- 如果 `allowRepeat = true` 且设置了 `repeatNodeId`
- 则跳转到重复对话节点
- 否则直接结束对话

## 🔄 兼容性说明

### 向后兼容

DialogManager 会自动判断数据格式：

**旧格式（简单对话）**
```csharp
DialogData
├── dialogPages: List<string>
└── dialogOptions: List<string>
```

**新格式（分支对话）**
```csharp
DialogData
├── nodes: List<DialogNode>
├── startNodeId: string
└── ...其他新字段
```

如果检测到 `nodes` 字段且有 `startNodeId`，则使用分支对话系统，否则使用简单对话系统。

### 迁移指南

**旧代码：**
```csharp
dialogData.dialogPages.Add("第一页");
dialogData.dialogPages.Add("第二页");
dialogData.dialogOptions.Add("选项 1");
```

**新代码：**
```csharp
var node = new DialogNode
{
    nodeId = "node_1",
    content = new List<string> { "第一页", "第二页" },
    options = new List<DialogOption>
    {
        new DialogOption
        {
            optionText = "选项 1",
            nextNodeId = ""
        }
    }
};
dialogData.nodes.Add(node);
dialogData.startNodeId = "node_1";
```
